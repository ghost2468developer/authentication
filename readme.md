Authentication API
A simple user authentication API built with FastAPI and JWT.

Features
User Registration

User Login

Profile Retrieval

List Users

Delete User

Update Password

Requirements
Python 3.9+

FastAPI

Uvicorn

Passlib

PyJWT

Installation

1. Clone the repository:
git clone https://github.com/yourusername/auth-api.git
cd auth-api
pip install -r requirements.txt

Running server
uvicorn main:app --reload
